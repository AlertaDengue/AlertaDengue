{% blocktrans %} É muito fácil descobrir se o seu município pode participar do Sistema de Alerta InfoDengue. Para isso é necessário que três etapas sejam cumpridas:

1 - Preencher o formulário de adesão com as informações sobre o município e sobre a pessoa responsável pelos dados de dengue com quem manteremos contato;

2 - Para dar continuidade à análise de viabilidade de participação, vamos precisar analisar os dados de notificação de dengue (SINAN) em seu município, de 2012 até o presente. Enviaremos para você uma carta sobre o compromisso do Sistema de Alerta InfoDengue a respeito da utilização dos dados. É importante ressaltar que não há necessidade de informação sobre a identificação dos casos.

3 - Se o seu município conseguiu preencher os dois requisitos anteriores, PARABÉNS! Os modelos de alerta já estarão prontos para seu município. Será elaborado e enviado um relatório contendo a análise dos dados históricos a você. Agora é hora de fazer o sistema funcionar daqui para frente. Para isso, um termo de compromisso deverá ser assinado entre o Sistema de Alerta InfoDengue e a Secretária Municipal de Saúde, autorizando a utilização dos dados.
